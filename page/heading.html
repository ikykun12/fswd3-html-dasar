HyperText Markup Language (HTML)

HTML merupkan standard bahasa yang digunakan untuk menampilkan document web. Berikut adalah beberapa hal-hal yang dapat dilakukan oleh HTML: Mengontrol tampilan dari web page dan contentnya. Membuat form Menambahkan object-object seperti image, audio, dan video. Dalam HTML, kita juga dapat menambahkan Cascading Style Sheets (CSS) ataupun JavaScript untuk menambahkan variasi pada konten yang akan kita tampilkan.
